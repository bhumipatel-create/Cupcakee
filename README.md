# Cupcakee
A curated collection of cupcake recipes from popular baking websites, featuring classic flavors, creative variations, and easy-to-follow instructions for beginners and baking enthusiasts.

🧁 Cupcake Recipe Website
📌 Project Overview


This project is an informative cupcake recipe website. The website provides a collection of cupcake recipes with an interactive and user-friendly interface.


It is designed to help users explore different cupcake flavors, view detailed recipes, and share their experiences.


✨ Features
        🏠 Home Page – Interactive cupcake gallery with hover effects
        🍰 Recipes Page – Browse cupcakes with links to detailed recipes
        📖 Recipe Page – Ingredients checklist, steps, comments, and image upload
        ℹ️ About Page – Project info with social media links
        📝 Sign Up Page – User registration with form validation
